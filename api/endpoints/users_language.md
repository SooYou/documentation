# users/language Endpoint

The users/language endpoint enables you to change your language settings.

### Endpoint

**users/language**

### available verbs

_PUT_

### Parameters

**language**: ISO 639-1 representation of the language you want to use

```json
{
    "language": "xx"
}
```

### Data returned

```js
{
    'data': [],
    'meta': {},
    'status': 'ok',
    'time': 'xx.xxxxxxxxxxx'
}
```
