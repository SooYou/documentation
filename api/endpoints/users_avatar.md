# users/avatar Endpoint

The users/avatar endpoint enables you to change your avatar.

### Endpoint

**users/avatar**

### available verbs

_PUT_

### Parameters

**id**: The avatarID of the avatar you want to use 

```json
{
    "id": "xxxxx"
}
```

### Data returned

```js
{
    'data': [],
    'meta': {},
    'status': 'ok',
    'time': 'xx.xxxxxxxxxxx'
}
```
