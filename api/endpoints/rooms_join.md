# rooms/join Endpoint

The rooms/join endpoint enables you to join a room.

### Endpoint

**rooms/join**

### available verbs

_POST_

### Parameters

**slug**: The URL conform name of the room you want to join

```json
{
    "slug": "xxxxx"
}
```

### Data returned

```js
{
    'data': [],
    'meta': {},
    'status': 'ok',
    'time': 'xx.xxxxxxxxxxx'
}
```
