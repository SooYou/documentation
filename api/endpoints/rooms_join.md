# rooms/join Endpoint

The rooms/join enables you to join a room

### Endpoint

**rooms/join**

### available verbs

_POST_

### parameters
```js
{
    'slug': 'xxxxx'         // The URL conform name of the room is needed to join``
}
```

### Data returned

```js
{
    'data': [],
    'meta': {},
    'status': 'ok',
    'time': 'xx.xxxxxxxxxxx'
}
```
