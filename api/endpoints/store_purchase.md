# store/purchase Endpoint

The store/purchase endpoint enables you to buy items from the store.

### Endpoint

**store/purchase**

### available verbs

_POST_

### Parameters

**id**: ID of the item you want to buy

```json
{
    "id": "xxx"
}
```

### Data returned

```js
{
    'data': [],
    'meta': {},
    'status': 'ok',
    'time': 'xx.xxxxxxxxxxx'
}
```
