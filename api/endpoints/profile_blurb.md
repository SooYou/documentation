# profile/blurb Endpoint

The profile/blurb endpoint enables you to change your profile message.

### Endpoint

**profile/blurb**

### available verbs

_PUT_

### Parameters

**blurb**: Your new profile message 

```json
{
    "blurb": "xxxxx"
}
```

### Data returned

```js
{
    'data': [],
    'meta': {},
    'status': 'ok',
    'time': 'xx.xxxxxxxxxxx'
}
```
