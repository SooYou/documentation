# playlists/:id/rename Endpoint

The playlists/:id/rename endpoint enables you to rename a playlist.

### Endpoint

**playlists/:id/rename**

### available verbs

_PUT_

### Parameters

**name**: The new name of the playlist

```json
{
    "id": -1
}
```

### Data returned

```js
{
    'data': [],
    'meta': {},
    'status': 'ok',
    'time': 'xx.xxxxxxxxxxx'
}
```
